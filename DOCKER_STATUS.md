# Docker Images Status

## Current Status: In Development

Docker images for DBLift are currently under development and will be published soon to GitHub Container Registry.

**Coming Soon:**
- `ghcr.io/dblift/dblift:latest` - Full DBLift image
- `ghcr.io/dblift/dblift:validation-latest` - Lightweight validation-only image

## Current Workaround

Until Docker images are published, please use one of these methods:

### Method 1: Download Pre-built Binaries (Recommended)

See [INSTALL.md](INSTALL.md) for platform-specific binaries.

### Method 2: Install from Source

```bash
git clone https://github.com/dblift/dblift.git
cd dblift
pip install -e .
```
